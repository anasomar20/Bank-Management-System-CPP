# 📖 Overview

**Bank Management System** is a console-based application developed in **C++** following **Object-Oriented Programming (OOP)** principles.

The project simulates a real banking system by providing secure authentication, client management, banking transactions, currency exchange, and role-based user permissions.

The application uses **file handling** for persistent data storage without relying on a database, making it an excellent demonstration of OOP, modular programming, and clean software architecture.
